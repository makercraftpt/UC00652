Entradas e saídas

  • BTN_1 — botão para ir ao 1º andar (cabina ou chamada exterior)
  • BTN_0 — botão para voltar ao R/C
  • SENSOR — sensor de chegada ao andar destino (1 = chegou)
  • P = Porta pode abrir (Moore: depende só do estado)
  • MS = Motor Sobe · MD = Motor Desce
Moore: P = Q₁' · Q₀' + Q₁ · Q₀' = Q₀' — a porta abre sempre que Q₀ = 0 (estados 00 e 10)

Self-loop — a Permanência é uma Transição

A regra de ouro: cada (estado, estímulo) TEM de ter um próximo estado definido
4 / 9
🔁 Self-loop — fico
O estímulo não foi satisfeito
→ próximo estado = estado atual
Exemplo:
Estou no PISO_0.
Ninguém carregou BTN_1.
→ Fico no PISO_0.
➡️ Mudança — movo
O estímulo foi satisfeito
→ próximo estado = outro estado
Exemplo:
Estou no PISO_0.
BTN_1 = 1 (chamada).
→ Passo a SUBINDO.

A regra da tabela

  • Para cada estado e para cada combinação de entradas, DEVE existir exactamente 1 próximo estado
  • Nenhuma combinação pode ficar "por definir" — a máquina tem sempre de saber o que fazer
  • O self-loop não é uma exceção ou um caso especial — é uma transição legítima como qualquer outra
  • Nos estados de movimento (SUBINDO, DESCENDO), o self-loop significa "ainda não chegaste — aguarda o SENSOR"
⚠️ Se uma linha da tabela ficar em branco, o circuito terá comportamento imprevisível — pode ir para um estado "lixo".

O elevador — quando fica, quando muda

🔁 Self-loops do elevador
  • PISO_0: fica se BTN_1 = 0 (ninguém quer subir)
  • SUBINDO: fica se SENSOR = 0 (ainda não chegou)
  • PISO_1: fica se BTN_0 = 0 (ninguém quer descer)
  • DESCENDO: fica se SENSOR = 0 (ainda não chegou)
➡️ Mudanças de estado
  • PISO_0 → SUBINDO: BTN_1 = 1
  • SUBINDO → PISO_1: SENSOR = 1
  • PISO_1 → DESCENDO: BTN_0 = 1
  • DESCENDO → PISO_0: SENSOR = 1
Repara: nos estados parado (PISO_0 e PISO_1), o estímulo é um botão. Nos estados de movimento, o estímulo é um sensor. Tipos diferentes de estímulo para situações diferentes.

Passo a Passo — Estado S0 (PISO_0)

Como preencher a tabela linha a linha · Uma linha por combinação de entradas
5 / 9

Método

1
Escolher o estado: S0 — PISO_0 (Q₁Q₀ = 00)
2
Listar todas as combinações de entradas relevantes.
S0 usa BTN_1 → 2 combinações: BTN_1=0 e BTN_1=1
3
Para BTN_1=0: "Ninguém quer subir. Fico aqui." → próximo = S0 (self-loop)
4
Para BTN_1=1: "Há chamada para o 1º andar. Arrancar." → próximo = S1 SUBINDO
5
Converter próximo estado em Q₁'Q₀' — os bits que os FF-D têm de guardar no próximo ciclo
Pergunta de controlo: "Se estou parado no PISO_0 e chega SENSOR=1, o que acontece?" — O sensor é irrelevante aqui: estou parado, não estou a chegar a lado nenhum. → self-loop também.

4 self-loops (roxo) + 4 mudanças = 8 linhas. Cada linha é uma decisão completa. Nenhuma combinação ficou por definir.

O que ler na tabela

  • Cada par (estado, condição) → 1 próximo estado. Sem ambiguidade.
  • As linhas self-loop têm Q₁'Q₀' = Q₁Q₀ (mesmos bits)
  • As linhas de mudança têm Q₁'Q₀' ≠ Q₁Q₀
  • As saídas P, MS, MD dependem só do estado atual (Moore) — não mudam ao longo da linha

Da Tabela às Equações

Q₁'Q₀' e saídas Moore — leitura directa das colunas
7 / 9

Saídas Moore — dependem só de Q₁Q₀

// Porta abre nos andares (S0=00, S2=10)
P = Q₁'·Q₀' + Q₁·Q₀' = Q₀'
// Motor sobe apenas em S1 (01)
MS = Q₁'·Q₀
// Motor desce apenas em S3 (11)
MD = Q₁·Q₀

Próximo estado — ler as 8 linhas da tabela

// D₁ = 1 nas linhas onde Q₁'=1 (S2 e S3)
D₁ = (Q₁'·Q₀·SENSOR) + (Q₁·Q₀'·1) + (Q₁·Q₀)
= Q₁'·Q₀·SENSOR + Q₁
// D₀ = 1 nas linhas onde Q₀'=1 (S1 self e S0→S1)
D₀ = (Q₁'·Q₀'·BTN_1) + (Q₁'·Q₀·SENSOR')
= Q₁'·(Q₀'·BTN_1 + Q₀·SENSOR')
Verificação rápida: quando Q₁Q₀=11 e SENSOR=1, D₁=1·1·1+1=1, D₀=0·(…)=0 → próximo = 00 = S0 ✓

Como derivar D₁ e D₀ da tabela

1
Olhar para a coluna Q₁' — marcar todas as linhas onde Q₁'=1
2
Para cada linha marcada, escrever o minterme: produto de Q₁Q₀ e entradas nessa linha
3
D₁ = soma (OR) de todos os mintermes com Q₁'=1
4
Repetir para D₀ (coluna Q₀')
5
Simplificar com álgebra booleana ou Karnaugh (TP07)

LogiSim — Implementar o Elevador

2 FF-D + 3 entradas + 3 saídas · verificar self-loops e mudanças
8 / 9

Passos

1
2 FF-D + Clock
Memory → D Flip-Flop (×2) + Clock. FF superior = Q₁, FF inferior = Q₀
2
3 entradas
Wiring → Pin (×3): BTN_1, BTN_0, SENSOR — ligar como entradas de 1 bit
3
Lógica D₁
D₁ = Q₁ + NOT(Q₁)·Q₀·SENSOR — portas OR, AND, NOT conforme a equação
4
Lógica D₀
D₀ = NOT(Q₁)·(NOT(Q₀)·BTN_1 + Q₀·NOT(SENSOR))
5
Saídas Moore → LEDs
P = NOT(Q₀) → LED verde · MS = NOT(Q₁)·Q₀ → LED azul · MD = Q₁·Q₀ → LED laranja
6
Testar self-loops
Com BTN_1=0: avançar clock — LED verde permanece aceso. Com BTN_1=1: LED azul (SUBINDO) acende → self-loop visível!
