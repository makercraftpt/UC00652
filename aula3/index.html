Combinatório vs. Sequencial

A diferença fundamental
3 / 9
⚡ Circuito Combinatório
Saída = f(Entradas atuais)
  • Portas lógicas simples
  • MUX / DEMUX
  • Somadores (74283)
  • Comparadores (7485)
Sem memória — se a entrada muda, a saída muda imediatamente.
🧠 Circuito Sequencial
Saída = f(Entradas, Estado anterior)
  • Latches
  • Flip-Flops
  • Contadores
  • Registos · FSM
Tem memória — Q mantém o valor até receber nova instrução.
💡 Um Latch é o elemento de memória mais simples — a base de tudo o que se segue.

⚠️ Estado Proibido (S̄=R̄=0) — Q e Q̄ ficam ambos a 1. Quando saímos deste estado, o resultado é imprevisível.

Latch SR com Portas NAND

Exemplo de uso real
5 / 9

Engenho de Furar — Acionamento com Memória

  • Botão START (ON) → S̄ = 0 → Set → motor liga
  • Botão STOP → R̄ = 0 → Reset → motor desliga
  • Entre os botões, o estado mantém-se — o motor lembra-se
  • Sem o latch, ao largar o botão o motor pararia
O circuito "lembra" a última ordem dada — isso é memória digital.

Latch D — Eliminar o Estado Proibido

Um inversor resolve o problema
6 / 9
S̄ = R̄ = 0 é agora impossível — o inversor garante que são sempre opostos.
ENDQ (próximo)Descrição
0XQ (mantém)Latch fechado
100Latch aberto
111Latch aberto
O Latch D elimina o estado proibido, mas Q segue D em tempo real enquanto EN=1 — é transparente.

Flip-Flop D com Relógio (Clock)

Disparo por flanco — sincronização precisa
7 / 9
Latch D — sensível ao nível
Q segue D sempre que EN = 1
→ Transparente, sem controlo preciso de tempo
Flip-Flop D — sensível ao flanco
Q captura D apenas no flanco ↑ do clock
→ Controlado, previsível, síncrono
Regra: Q atualiza para o valor de D apenas no flanco ↑ do clock. Entre flancos, Q não muda — mesmo que D mude.

Cronograma Temporal (Timing Diagram)

FF-D com disparo no flanco positivo ↑ — Q inicial = 0
8 / 9
Q captura o valor de D no momento exato do flanco ↑ — qualquer mudança de D entre flancos não afeta Q.
